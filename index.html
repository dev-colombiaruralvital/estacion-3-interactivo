¡Bienvenidos a la Caja de Herramientas para certificar muertes ocurridas en domicilio! Aquí, te proporcionamos información y recursos esenciales para la certificación de este tipo de defunciones. Así, inicialmente, encontrarás un recurso con las principales consideraciones que se deben tener en cuenta para el proceso de certificación. También, hemos diseñado un interactivo con los signos y síntomas clave que pueden ayudarte a identificar circunstancias especiales alrededor de una defunción y finalmente, encontrarás el manual del certificador y el formulario de antecedente médico de defunción.

Te recomendamos comenzar por estudiar el documento base y luego, explorar cada recurso de manera detallada para fortalecer tus conocimientos en este importante ámbito.
